# Backend - Expense Tracker Starter

## Quick start
1. Copy `.env.example` to `.env` and set your `MONGO_URI` and `JWT_SECRET`.
2. Install dependencies:
   ```
   npm install
   ```
3. Start dev server:
   ```
   npm run dev
   ```
4. API endpoints:
   - POST /api/auth/register
   - POST /api/auth/login
   - GET /api/expenses
   - POST /api/expenses
   - PUT /api/expenses/:id
   - DELETE /api/expenses/:id
